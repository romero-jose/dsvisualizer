
dsvisualizer
=====================================

Version: |release|

A simple data structure visualizer for Jupyter Notebooks


Quickstart
----------

To get started with dsvisualizer, install with pip::

    pip install dsvisualizer

or with conda::

    conda install dsvisualizer
